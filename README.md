# SQI College of ICT Website Clone

## Overview

This project is a front-end clone of the SQI College of ICT website (sqi.edu.ng), created as part of my web development learning journey and portfolio projects.

The goal of this project was to improve my ability to analyze, recreate, and implement a professional website design while applying modern web development best practices.

**Disclaimer:** This project was developed solely for educational and portfolio purposes. All original branding, content, and design inspiration belong to SQI College of ICT.

---

## Features

* Fully responsive design
* Mobile-first approach
* Clean and organized code structure
* Modern user interface implementation
* Interactive navigation components
* Optimized layout for different screen sizes
* Recreated key sections of the original website
